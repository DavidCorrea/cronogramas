# src — Application source

This folder contains all application code.

| Folder | Purpose |
|--------|---------|
| **app/** | Next.js App Router: pages (routes) and API route handlers. See **app/context.md** and **docs/CLIENT.md** (pages), **docs/API.md** (API). |
| **components/** | Shared React components (nav, forms, schedule view). See **components/context.md**. |
| **db/** | Drizzle schema and migrations. See **db/context.md** and **docs/DATABASE.md**. |
| **lib/** | Shared utilities: auth, API helpers, scheduler, Zod schemas, group context, etc. See **lib/context.md**. |

For product behaviour, scripts, and agent rules see **AGENTS.md** at the project root.
